# inter_ web for idd master (art)

2023年度 多摩美術大学大学院デザイン専攻情報デザイン領域 有志展「Inter_」（フューチャーラボ／イメージラボ／メディアラボ）のためのウェブサイト

## npm でスタートする

```shell
$ git clone https://github.com/akisoqls/inter__web.git
$ cd inter__web
$ npm install
$ npm run dev
```
